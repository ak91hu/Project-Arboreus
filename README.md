**The Arboreus Algorithm** is an educational puzzle game designed to visualize the logic behind Binary Search Trees (BST) and Red-black trees. Built with Java and LibGDX, it challenges players to construct a valid data structure in real-time under constraints, blending gamification with algorithmic learning.

This project uses Gradle for dependency management. The wrapper is included, so no manual Gradle installation is required.

### Prerequisites
- **Java JDK 17** or higher.

### Running the Game

1. **Clone the repository:**
   ```bash
   git clone https://github.com/ak91hu/Project-Arboreus.git
   cd Project-Arboreus
   ```

2. **Run via Terminal:**

   **Windows:**
   ```powershell
   gradlew desktop:run
   ```

   **Linux / macOS:**
   ```bash
   ./gradlew desktop:run
   ```
